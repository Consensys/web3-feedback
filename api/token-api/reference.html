Linea Token API (0.1.0)

Download OpenAPI specification:Download

API to access token information on Linea, including prices, most traded tokens, top gainers/losers over 24h. Each endpoint can optionally include price data.

Tokens

Get tokens

Retrieve a list of tokens based on the provided query parameters

query Parameters
page
number >= 1
Default: 1

Page number

limit
number [ 1 .. 100 ]
Default: 50

Number of items per page

isSecure
boolean

Filter tokens secure tokens

isMostSwapped
boolean

Filter tokens that have been most swapped

new
boolean

Filter tokens that have been newly added

addresses
string

Comma-separated list of tokens addresses (comma-separated)

order
string (OrderColumns)
Enum: "name" "symbol" "decimals" "contractAddress" "currentPrice" "priceUpdatedAt" "createdAt" "updatedAt" "last24hVariation" "last24hVariationAbsolute" "sells" "buys" "isPossibleSpam" "totalSupply" "fdv"

Order by column

sort
string
Enum: "asc" "desc"

Order sort (asc, desc)

Responses

https://token-api.devnet.linea.build/tokens
